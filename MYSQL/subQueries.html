Sub Queries/Nested Queries

Sub Queries/Nested Queries

  • A subquery is a select query that is contained inside another query. The inner select query is usually used to determine the results of the outer select query.
  • The output of the subquery is given as input to the main query such that main queryis dependent on subquery.
  • Subquery or Inner query or nested query is a query within another SQL query and embedded within the where clause•Should be enclosed within parenthesis
  • A subquery can have only one column in the select clause unless multiple columns are in the main query for the subquery to compare its selected columns.
  • Order by cannot be used in subquery although the main query can use order by.•Subqueries thatreturn more than one row can be used with multiple value operators, such as the IN operator.

Types of subquery

1.Single row subqueries.
2.Multiple row subqueries.

Single row subquery: if the subquery or inner query returns exactly one record we call it as a single-row subquery. (we can use normal operators)

Multi-rowsubquery:if the subquery or inner query returns more than one record we call it as a multi-row subquery. (we can use special operators IN, NOT IN, ALL, ANY)

WAQ to display the details of employees working in California.

Solution: select*
from empdetails
where worksin=(select roid
from retailoutlet

WAQ to display the details of items purchased by John from the retail stock.

Solution: select *
from retailstock
where itemcode=(select itemcode
from purchasebill
where custid=(select custid
from customer vwherecustname='John'));

WAQ to display the customer id, customer name and email id of customers who have purchased in New York outlet.

Solution: select custid,custname,emailid
from customer
where custid in(select custid
from purchasebill
where roid=(select roid from retailoutlet
where location='New York'));

WAQ to find the customer id who have purchased Lee T-Shirt

Solution: select custid
from purchasebill
where itemcode in(select itemcode
from item
where descr='Lee T-Shirt');
