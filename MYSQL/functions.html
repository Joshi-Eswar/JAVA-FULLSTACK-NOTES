FUNCTIONS

FUNCTIONS



InMySQL, we have two types of functions:
1.Single row functions
2.Multiple row functions

Single row functions

Are such functions which will accept a single row as input (or) it accepts multiple rows as input but produces one result per row.Single row functions have four types:
1.String Function
2.Date Function
3.Number Function
4.General Function

Multiple row functions/group functions/aggregate function

are such functions which will accept a single row or multiple rows as input but produces one result per group.

Case Manipulation Function


Upper:Converts a string to upper-case.
Syntax: select upper(string)
Example: SELECT UPPER ("HiGoodMorning");
Output:HI GOODMORNING

Lower:Converts a string to lowercase
.Syntax: select lower(string)
Example: SELECT lower ("HI GOODMORNING");
Output:higoodmorning

WAQ to display employee name in uppercase and email id in the lower case from employee details table only if they work as Administrator.
Solution:SELECT upper(empname),lower(emailid)
from empdetails
where designation='Administrator';

WAQ to display the description of an item in uppercase only if item type is FMCG.
Solution:SELECT upper(descr)
from item
where itemtype='FMCG';

Character Manipulation Function

Length
Length is used to return the length of the string (in characters).
Syntax: select length(string);
Example: select length(‘ABC FOR TECHNOLOGY’);
Output: 18

WAQ to display the length of store name from quotation table only if the quotation status is accepted.
Solution: select length(sname)
from quotation
where qstatus='Accepted';

WAQ to display the email ID and the length of mail id from customer table only if customer type is regular. Sort the result based on the customer id in ascending order.
Solution: select emailid,length(emailid)
from customer
where custtype='Regular'
order by custid;

Reverse

It is used to reverse the string and returns the result.
Syntax: select reverse(string);
Example: select reverse('RajaRamRohanRoy');
Output:yoRnahoRmaRajaR

WAQ to display customer name from customer table only if customer name is a palindrome.
Solution: select custname
from customer
where custname=reverse(custname);

Concat

It is used to concat two or more strings into a single string
Syntax: select concat(expression1, expression2..);
Example:select concat('john','jacobs');
Output:johnjacobs

WAQ to display item type and item description while displaying provide an alias name as "ITEM DESCRIPTION."
Solution:select concat(itemtype,' ',descr) AS 'ITEM DESCRIPTION'
from item;

Replace

Replaces all occurrences of a substring within a string, with a new substring.
Syntax:
select replace(string,search_string,new_string);

WAQ to display the customer type from Regular to Prime using the customer table.
Solution: select replace(custtype,'Regular','Prime')
from customer;

WAQ to display employee name and designation of those employees who works as Administrator.While Displaying replace Administrator to Admin Manager.
Solution:select empname,replace(designation,'Administrator','AdminManager')
from empdetails
where designation= 'Administrator';

Substr

Substr function it is used to extract a substring from a string.
Syntax: select substr(string,start_position, length)
Example: select substr('Hello Good Morning',7,4);
Output:Good

WAQ to display the last three digits of the item code from item table and give alias name as code. Finally, sort the result in descending order.
Solution: selectsubstr(itemcode,-3) as code
from item
order by 1desc;

WAQ to display the customer name and the first character of the customer name from the customer table while displaying the result, display the name along with the first character.
Solution: select concat(custname,' ',substr(custname,1,1))
from customer;

WAQ to display the description of an item in the upper case from item table only if the description starts with a character 'b'.
Solution: select upper(descr)
from item
where substr(descr,1,1)='b';

Instr

>Instrfunction returns the position of the first occurrence of a string.
Syntax: selectinstr(string, search string);
Example: Select instr('sachin','a');
Output:2

WAQ to display the position of character 'a' in customer name from the customer table.
Solution: selectinstr(custname,'a')
from customer;

Trim

Trim function removes the unwanted leading and trailing characters from a string.
Example:select trim(trailing 'm' from 'madam');
Output:mada

select trim(leading 'm' from 'madam');
Output:adam

select trim(both 'm' from 'madam');Output:ada


Ltrim

Ltrimfunction is used to remove all leading spaces of a string.
Syntax:select ltrim(string);
Example:selectltrim('ABC');

Rtrim

Rtrim function is used to remove all trailing spaces of a string.
Syntax: select rtrim(string);
Example:selectrtrim('ABC');

Lpad

Lpad function left-pads a string with another string, to a certain length.
Syntax:select lpad(string, length, lpad_string);
Example:selectlpad('demo',10,'*');
Output:******demo

Rpad

Rpad function right-pads a string with another string, to a certain length.
Syntax: select rpad(string, length, rpad_string);
Example:selectrpad('demo',10,'*');
Output:demo******

WAQ to display employee name and updated contact number by right padding with '#' of 10 character length field and give an alias name "updated contact".
Solution: select empname,rpad(contactno,'10','#') as "updated contact"
from empdetails;

Left

Left function extracts the specified number of characters from a string starting from the left side.
Syntax: select LEFT(string, number_of_chars);
Example:select left('www.abcfortech.com',3);
Output: www


Right

Right function extracts the specified number of characters from a string starting from the right side.
Syntax: select right(string, number_of_chars);
Example:select right('www.abcfortech.com',3);
Output: com


WAQ to display the last three digits of an employee id and first three characters of employee name from the employee table and give an alias name asnamecode.
Solution:select concat(right(empid,3),left(empname,3)) as Name Code
from empdetails;

ASCII

ASCII function returns the ASCII value for the specific character.
Syntax: select ascii(charater);
Example: select ascii('a');
Output:97


Substring_index

This function returns a substring of a string before a specified number of delimiter occurs.
Syntax: select SUBSTRING_INDEX(string, delimiter, number);
Example: SELECT SUBSTRING_INDEX("www.abcfortech.com", ".", 2);
Output:www.abcfortech

NUMBER FUNCTIONS



MOD

>Mod function returns the remainder of a number divided by another number.
Syntax: select MOD(x, y);

Example: select mod(20,2);
Output:0

WAQ to display Only Odd Rows From employee details table.
Solution: select *
from empdetails
where mod(empid,2)=1;


ROUND

Round function rounds a number to a specified number of decimal places
Syntax: select ROUND(number, decimals);
Example: select round(3456.23788, 2);
Output: 3456.24


WAQ to round off the number 45.326 at 2nd decimal place.
Solution: Select round (45.326,2);

Write the output for the following
Round(2678.345,-2)

TRUNCATE

>Truncate function truncates a number to the specified number of decimal places.
Syntax: select TRUNCATE(number, decimals);

Example: select truncate(3456.23788,2);
Output: 3456.23


WAQ to truncate the number 45.326 to 2nd decimal place.
Solution: Select truncate(45.326,2);


Write the output for the following Truncate(2678.345,-2)

POWER or POW

Power function returns the value of a number raised to the power of another number.
Syntax: select POWER(x, y);

Example: select POWER(2,4);

Write a query to find the power of 5^2
Solution: select power(5,2);


SQRT

SQRT function returns the square root of a number.
Syntax: select SQRT(number);

Example: select SQRT(25);

Write a query to find the square root of 4.
Solution: select sqrt(4);


ABS

>ABS function returns the absolute (positive) value of a number.
Syntax: ABS (number);
Example: SELECT ABS(-243.5);
Output: 243.5


CEIL

>CEIL function returns a nearest highest integer value.
Syntax: select CEIL(number);

Example: SELECT CEIL(25.75);
Output:26


FLOOR

Floor function returns a nearest lowest integer value.
Syntax: select floor(number);
Example: SELECT floor(25.75);
Output:25

DATE AND TIME FUNCTION



DATE

Date function extracts the date part from a DateTime expression.
Syntax: DATE(expression);

Example: SELECT DATE('2019-06-13 19:30:10');

TIME

Time function extracts the time part from a date/time expression.Syntax: TIME(expression);

Example: SELECT TIME('2019-06-13 19:30:10');

GENERAL FUNCTIONS



Ifnull

Ifnullfunction returns a specified value if the expression is NULL.
Syntax: select IFNULL(expression, alt_value);


WAQ to display payment date from the orders table. If the payment date is null, then display the payment date as "not paid".

Solution: select ifnull(pymtdate, 'not paid')
from orders;


AGGREGATE FUNCTION OR MULTIPLE ROW FUNCTION

Min function is used to return the minimum value from the set of values.
Max function is used to return the maximum value from the set of values.
Count function is used when you need to count some records of the table.
Sum function is used to return the total summed value from the set of values.
Avg function is used to return the average value from the set of values.

WAQ to display totalprice,minimum price,maximum price,average price and also total number of items from item table.
Solution: select sum(price),min(price),max(price),avg(price),count(*)
from item;


WAQ to display the number of employees having a contact number. Give an alias name as "count of emp with contact_no".
Solution: select count(*) "count of emp withcontact_no"
from empdetails
where contact no is not null;
