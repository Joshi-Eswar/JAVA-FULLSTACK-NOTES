ORDER BY

To sort the result set in ascending order or descending order we use "Order by" clause.

If you use MYSQL ORDER BY clause without specifying the ASC and
DESC modifiers then by default you will get the result in ascending Order.

Syntax:
SELECT expression
FROM tables
[WHERE condition]
ORDER BY expression [ASC|DESC];

WAQ to display the customer's name in Alphabetical Order.
Solution:select custname
from customer
order by custname;

WAQ to display the item code and the quantity purchased. Display the quantity such that the highest quantity is on top.

Solution:select itemcode,quantity
from purchasebill
order by quantity desc;

WAQ to display customer id, bill amount, and the quantity purchased. While displaying sort the result by customer id then by the bill amount.

Solution:select custid,billamount,quantity
from purchasebill
order by custid,billamount;
