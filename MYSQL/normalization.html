Spurious tuple
They are such tuples which are not present in the original table but gets generated by decomposing and further join.

Anomalies - Problems
Insertion anomaly, Deletion Anomaly, Updation Anomaly

Data redundancy
The repetition of data inside the table which leads to memory wastage is called as data redundancy

Normalization
The systematic approach of decomposing the tables in order to eliminate anomalies is referred to as normalization.
