GROUP BY CLAUSE



The GROUP BY clause is used to group rows that have the same values.

The GROUP BY clause returns one row for each group. In other words, it reduces the number of rows in the result set.

Syntax:
SELECT col1,col2,col3,,,coln,aggregate function
FROM table
WHERE condition
GROUP By col1,col2,col3,,,coln;

We can use group by clause without where clause.

WAQ to display distinct item type from item table without using the distinct keyword.
Solution: select itemtype
from item
group by itemtype;

WAQ to display designation and the minimum salary in each designation. Give an alias name to the minimum salary as 'minimum salary'.
Solution: select designation,min(salary) as "minimum salary"
from empdetails
group by designation;


WAQ to display the total number of male customers and the total number of female customers from the customer table.
Solution: select gender,count(*)
from customer
group by gender;

WAQ to display customer id and total bill amount paid by the customer from purchase bill table,Only if the customer id is 2002 or 2003. Finally, sort the result in such a way that the highest bill amount should be the first.
Solution: select custid,sum(billamount)
from purchasebill
where custid=2002 or custid=2003
group by custid
Order by sum(billamount)desc;
