Having CLAUSE

Having CLAUSE



Having clause is used to filter the group.

Difference between WHERE and HAVING

Where:
WHERE is used to filter rows before grouping.
WHERE clause cannot contain aggregate function likeCOUNT(),SUM(),MAX(),MIN(),AVG().

Having:
HAVING is used to filter records after grouping.
HAVING clause may contain aggregate functions.

WAQ to display the designation and minimum salary in each designation but the minimum salary should be greater than 2500.
Solution: select designation,min(salary)from empdetails group by designation having min(salary)>2500;

WAQ to display the item code and total unit price for each item code only if the total unit price is greater then 300 from the retail stock table.
Solution:
select itemcode,sum(unitprice)
from retailstock
group by itemcode
having sum(unitprice)>300;

UNION

•The SQL Union operation is used to combine the result of two or more SQL SELECT queries.
•In the UNIONoperations,thedata type and columns must be same in both the tables on which UNION operation is being applied.
•The Union operationeliminates the duplicate rowsits result set.


Syntax:
SELECT column_name FROM table1
UNION
SELECT column_name FROM table2;

UNION ALL
